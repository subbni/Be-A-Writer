# Be-A-Writer
## 프로젝트 소개
사용자가 자유롭게 글을 작성하고 저장할 수 있는 간단한 글쓰기 플랫폼입니다.

개인 프로젝트로 진행되었으며, 백엔드와 프론트엔드 모두 직접 구현하였습니다.
- Backend: `Node.js` + `Express`
- Frontend: `React`
